## 📊 Employee Insights Dashboard

### 📌 Project Overview

This project analyzes employee data to uncover insights related to workforce distribution, hiring trends, salary patterns, and gender diversity. The dashboard provides a clear and interactive view of organizational data.

---

### 🔗 Live Dashboard

View the interactive dashboard here:
https://public.tableau.com/app/profile/simarjot.kaur8544/viz/EmployeeInsightsDashboard_17768581101140/EmployeeInsightsDashboard

---

### 📂 Dataset

Dataset sourced from Kaggle:
https://www.kaggle.com/datasets/crepantherx/employees

---

### 🛠️ Tools Used

* MySQL → Data cleaning, joins, and transformation
* Tableau → Data visualization and dashboard creation

---

### 🔧 Steps Performed

* Imported raw dataset from Kaggle
* Performed joins across multiple tables (Employees, Departments, Salaries)
* Created a clean final dataset using SQL
* Exported dataset for visualization
* Built an interactive dashboard in Tableau

---

### 📈 Key Insights

* Hiring trends over time
* Department-wise employee distribution
* Salary distribution across employees
* Gender distribution of employees
* Average salary comparison across departments
